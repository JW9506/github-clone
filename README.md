## Features

- github clone using

  1. NextJS
  2. koa
  3. @reduxjs/toolkit
  4. antd

## Getting Started

### create `.env.development.local`

- provide `client_id` and `client_secret` which can be obtained for free in `Settings` > `Developer settings` in Github

### then

```bash
yarn dev
```
